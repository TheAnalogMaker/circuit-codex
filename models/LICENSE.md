# Tube model license — CC0 1.0

Every SPICE model under `models/` is dedicated to the public domain under
**CC0 1.0 Universal**: <https://creativecommons.org/publicdomain/zero/1.0/>

Use them anywhere, for anything, no attribution required (though a link back is
appreciated).

**Provenance requirement**: models in this directory must be *freshly derived* —
parameter fits produced from published tube datasheet curves (plate characteristics,
transfer curves) using documented methodology. Do **not** contribute models copied
from other collections (Duncan Amp Pages, Norman Koren's sets, vendor libraries, etc.);
those carry their own per-model terms and cannot be re-licensed here. Each model file
must state its datasheet source and fit method in a header comment.
